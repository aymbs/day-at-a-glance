# Day at a Glance

Personal intentional living dashboard with daily tasks, habits, goals, workouts, books, journal, and more.

## Features
- Daily/Weekly/Monthly views with date navigation
- Habit tracking with consistency heatmap
- Workout tracker with pie chart visualization
- Book tracker with Goodreads import
- Focus timer (Pomodoro)
- Journal with past entry browsing
- Floating sticky notes
- Google Calendar integration
- Data syncs across devices via Supabase

## Tech Stack
- Next.js
- Supabase (PostgreSQL)
- Deployed on Vercel
